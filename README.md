# AML — 6D Pose Estimation on LineMod

RGB-D 6-DoF object pose estimation pipeline: YOLO detection → pose regression via ResNet-50 backbone (+ depth branch for RGB-D variants).

## Setup

### 1. Install PyTorch (CUDA)

```bash
pip3 install torch torchvision --index-url https://download.pytorch.org/whl/cu130
```

On macOS (MPS) or CPU-only, install without the index URL.

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

> **Note:** `requirements.txt` may contain Windows/CUDA-specific wheel URLs. On macOS, install packages individually or skip torch from requirements (already installed above).

### 3. Download dataset and weights

```bash
bash download_data_and_weights.sh
```

This downloads LineMod preprocessed (~1 GB), unpacks it to `datasets/linemod/Linemod_preprocessed/`, downloads all model weights to `weights/`, and runs `phase2_detection/prepare_yolo_data.py` to generate the YOLO format dataset.

## Running the pipeline

### Phase 2 — YOLO Detection

```bash
# Prepare YOLO-format dataset (already run by download script)
python -m phase2_detection.prepare_yolo_data

# Train YOLO11n on LineMod
python -m phase2_detection.train

# Evaluate detection (mAP on test split)
python -m phase2_detection.evaluate
```

### Phase 3 — Baseline RGB-only

```bash
python -m phase3_baseline.train
python -m phase3_baseline.evaluate
python -m phase3_baseline.visualize
```

### Phase 4 — RGB-D Fusion (main: ResNet-50 + ResNet-18)

```bash
python -m phase4_fusion.main.train
python -m phase4_fusion.main.evaluate
python -m phase4_fusion.main.visualize
```

### Phase 4 — RGB-D Fusion (extension: ResNet-50 + custom ResNet-10)

```bash
python -m phase4_fusion.extension.train
python -m phase4_fusion.extension.evaluate
python -m phase4_fusion.extension.visualize
```

## Device selection

All scripts auto-select CUDA → MPS → CPU in order of availability.
